Portfolio

Training For Gains
A work out logging application created using Ruby on Rails backend and a NodeJS/AngularJS front end. I created the app as a gym enthusiast who sees people writing their workout routine at a gym when they can lose the notepad and have it be entirely mobile. This way, the app can store the data without every worrying about forgetting a notepad to the gym.

Where's My Kogi
Still in its early stages. Where's My Kogi is a web application built using NodeJS, MongoDB, Express. It is a Korean bbq tracker that logs the user's restaurant visited along with some information about the restaurant. I created it with the purpose of getting users a chance to keep track of Korean bbq as well as to provide some information about what a Korean bbq usually entails. Check out to log your newly discovered Kbbq restaurants!

What's Cooking
A web application designed for weekly food planning to lead a healthy lifestyle. It is built using a full MEAN stack (MongoDB, Experss, Angular, and NodeJS). A group project that allowed us to experience a remote group interaction using Github repos to push/pull and merge code for collaboration purposes. It is a project that allowed me to appreciate github technologies in relation to version control.

Topshelf Imaginations
A two day MEAN stack project that is able to create a full CRUD for a library collections of all the users in the site. I created a simplified library collection that is simple in looks as it logs the data as if the user is writing on a notebook with the tab styling in the front end.
